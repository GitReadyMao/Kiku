■書体概要
見出しやタイトルなど目立たせたいところに向いたオールドスタイルの明朝体です。伝統的な築地初号の流れを汲む、「築豊初号見出しかな」を元にブラッシュアップされた仮名とどっしりとした骨のある筆脈が特徴です。
漢字には角立て処理の施されている源ノ明朝の派生フォント「源流明朝2.100」を利用しています。
オリジナルのフォントに深く感謝いたします。
